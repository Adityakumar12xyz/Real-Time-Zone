## ⚡ Core Capabilities

*   **🌍 Global Synchronization:** Instantly tracks and displays accurate time across multiple critical time zones (New York, London, Tokyo, IST, etc.).
*   **⏱️ Zero-Latency Ticking:** Powered by highly optimized Vanilla JavaScript `setInterval` logic that prevents UI lag.
*   **🌙 Dynamic Day/Night Cycle:** The interface subtlely adapts to represent day or night states of the target time zones.
*   **📅 Precision Date & Day Tracking:** Doesn't just show the time—it calculates and displays the exact day and date shift across the globe.
*   **🚀 Ultra-Lightweight:** Zero heavy libraries. Pure HTML, CSS, and JS execution for instant load times.
